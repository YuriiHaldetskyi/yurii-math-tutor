# Лендинг Юрія Галдецького

Готовий статичний сайт репетитора з математики.

## Перегляд

Відкрийте `index.html` у браузері. Для публікації завантажте весь вміст папки на хостинг без зміни структури `assets`.

## Основні файли

- `index.html` — структура і тексти;
- `styles.css` — дизайн та адаптивна верстка;
- `script.js` — мобільне меню, анімації та карусель;
- `assets` — фотографії, сертифікати й захищені водяним знаком приклади workbook'ів.

Кількісні показники досвіду й результатів на сайті наведені за даними Юрія Галдецького.
